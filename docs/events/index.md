# Events

Events are small chunks are data related to the application that need to be
handled correctly.

They can be of different categories:

| Name                       | Category                            |
|--------------------------- | ----------------------------------- |
| Key events                 | `BRASIO_EVENT_CATEGORY_KEY`         |
| Mouse events               | `BRASIO_EVENT_CATEGORY_MOUSE`       |
| General Application events | `BRASIO_EVENT_CATEGORY_APPLICATION` |
